Ejercicio 1:

Ejercicios para realizar en Local Storage / Session Storage:

Realiza un script para que cuando el usuario pulse el botón “Guardar datos”, se almacenen en el navegador los datos introducidos. Cuando pulse “Mostrar datos”, realizará una búsqueda del usuario cuyo DNI se indique y se mostrarán los datos de dicho usuario guardados. Cuando pulse “Borrar registro”, borrará el usuario con el DNI indicado. Si se pulsa “Borrar todos los registros”, se borrarán todos los registros guardados.

Ejercicio 3:

Ejercicios para realizar en Local Storage / Session Storage:

Realizar un script para que al pulsar sobre el botón “Añadir monumento” del siguiente formulario, se guarde en el navegador el nombre del monumento, el país donde está ubicado y una URL donde se encuentra una imagen del monumento. En el buscador de monumentos al escribir el nombre del documento y pulsar “Mostrar datos”, se añadirán debajo el nombre y el país del monumento. Al pulsar “Borrar monumento” se borrará el registro del monumento y, al pulsar ”Borrar todos”, se borrarán todos los datos guardados en el navegador.

Buscador de monumentos
