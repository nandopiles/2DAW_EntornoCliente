Actividad 1:

Implementa un script que permita validar el siguiente formulario: el nombre y los apellidos deben estar compuestos por letras y el DNI debe de ser un número de ocho números. Al validar el DNI se calculará la letra del DNI. Si se introduce un dato incorrecto, saltará una alerta indicando que vuelva a introducirlo. Si todo es correcto, saltará una alerta indicando la letra del DNI.

Actividad 2:

Implementa un script que permita validar el siguiente formulario: el nombre y los apellidos deben estar compuestos únicamente por letras y el correo electrónico introducido debe ser correcto. Si se introduce un dato incorrecto, se borrará el campo del formulario incorrecto y saltará una alerta indicando que vuelva a introducirlo.

Actividad 3:

Implementa un script que permita validar el siguiente formulario. Una vez validado, sustituye este enunciado con la reseña y los datos introducidos por el usuario.
